**Como tirar bons prints:**
1. Rode o sistema localmente: `python app.py`
2. Use F11 para tela cheia no navegador
3. Ctrl+Shift+I → Toggle device toolbar para simular celular
4. Use a extensão "Full Page Screen Capture" para capturar a página inteira
